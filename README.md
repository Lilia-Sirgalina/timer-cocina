## Project Description

This application allows users to set a countdown timer.  
The app is built using HTML, CSS, and JavaScript.  

Enter the desired number of minutes in the **"Minutes"** field and click **"Start"** to begin the countdown.  
The timer will start counting down, and when it reaches zero, a sound alert will play.  
To stop and reset the timer, click the **"Stop/Reset Timer"** button.  

No installation is required.

### How to Use:
1. Enter the number of minutes in the **"Minutes"** field.
2. Click the **"Start"** button.
3. The timer will begin counting down. A sound alert will play when the countdown ends.
4. To cancel and reset the timer, click the **"Stop/Reset Timer"** button.

### Features:
- Countdown timer functionality using JavaScript.
- If the user tries to start the timer without entering minutes, a warning message will be displayed.
- Intuitive user interface.

## Requirements
- A modern web browser with JavaScript support.
